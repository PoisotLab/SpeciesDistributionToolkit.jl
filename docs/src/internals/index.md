# Documentation of integration functions

These manual pages are focused on the functions integrating the different
components of `SpeciesDistributionToolkit` together, as well as providing
information about how to contribute to the package.

As a user, virtually everything you need to know is either in the how-to, the
tutorials, or the docstrings of one of the functions used in either of these.

The full docstrings for the different packages are in their own documentation
webpages, which are accessible from the "Ecosystem" tab in the top menu.
