Use with the SDeMo package ​

In this tutorial, we will reproduce the excellent tutorial on SDMs by Damaris Zurell, using the same species and location, a similar dataset, but a different algorithm.

julia
using SpeciesDistributionToolkit
using CairoMakie
using Statistics
using Dates

Note that this tutorial is not showing all the capacities of the SDeMo package!

Getting the data ​

julia
CHE = SpeciesDistributionToolkit.gadm("CHE");

In order to simplify the code, we will start from a list of bioclim variables that have been picked to optimize the model:

julia
bio_vars = [1, 7, 10]
3-element Vector{Int64}:
  1
  7
 10

We get the data on these variables from CHELSA2:

julia
provider = RasterData(CHELSA2, BioClim)
layers = [
    SDMLayer(
        provider;
        layer = x,
        left = 0.0,
        right = 20.0,
        bottom = 35.0,
        top = 55.0,
    ) for x in bio_vars
];

And we then clip and trim to the polygon describing Switzerland:

julia
layers = [trim(mask!(layer, CHE)) for layer in layers];
layers = map(l -> convert(SDMLayer{Float32}, l), layers);

The next step is to get the data, using the eBird dataset:

julia
ouzel = taxon("Turdus torquatus")
presences = occurrences(
    ouzel,
    first(layers),
    "occurrenceStatus" => "PRESENT",
    "limit" => 300,
    "datasetKey" => "4fa7b334-ce0d-4e88-aaae-2e0c138d049e",
)
while length(presences) < count(presences)
    occurrences!(presences)
end

And after this, we prepare a layer with absences. Note that this code is not particularly beautiful but will be fixed when we release the occurrences interface package as part of a next version.

julia
presencelayer = zeros(first(layers), Bool)
for occ in mask(presences, CHE)
    presencelayer[occ.longitude, occ.latitude] = true
end

The next step is to generate a pseudo-absence mask:

julia
background = pseudoabsencemask(DistanceToEvent, presencelayer)
bgpoints = backgroundpoints(background, sum(presencelayer))
SDM Layer with 69967 Bool cells
	Proj string: +proj=longlat +datum=WGS84 +no_defs
	Grid size: (239, 543)

We can take a minute to visualize the dataset, as well as the location of presences and pseudo-absences:

Code for the figure
julia
f = Figure(; size = (600, 300))
ax = Axis(f[1, 1]; aspect = DataAspect())
heatmap!(ax,
    first(layers);
    colormap = :linear_bgyw_20_98_c66_n256,
)
scatter!(ax, presencelayer; color = :black)
scatter!(ax, bgpoints; color = :red, markersize = 4)

Setting up the model ​

These steps are documented as part of the SDeMo documentation. The only difference here is that rather than passing a matrix of features and a vector of labels, we give a vector of layers (features), and the two layers for presences and absences:

julia
sdm = SDM(MultivariateTransform{PCA}, NaiveBayes, layers, presencelayer, bgpoints)
MultivariateTransform{PCA} → NaiveBayes → P(x) ≥ 0.5

Out of curiosity, we can check the expected performance of the model:

julia
folds = kfold(sdm);
cv = crossvalidate(sdm, folds; threshold = true);
mean(mcc.(cv.validation))
0.6733633901602248

We will now train the model on all the training data:

julia
train!(sdm)
MultivariateTransform{PCA} → NaiveBayes → P(x) ≥ 0.466

Making the first prediction ​

We can predict using the model by giving a vector of layers as an input. This will return the output as a layer as well:

julia
prd = predict(sdm, layers; threshold = false)
SDM Layer with 69967 Float64 cells
	Proj string: +proj=longlat +datum=WGS84 +no_defs
	Grid size: (239, 543)

To get a sense of the variability in the outcome, we will do a quick bootstrap aggregating with 50 different bags:

julia
ensemble = Bagging(sdm, 50)
train!(ensemble)
unc = predict(ensemble, layers; consensus = iqr, threshold = false)
SDM Layer with 69967 Float64 cells
	Proj string: +proj=longlat +datum=WGS84 +no_defs
	Grid size: (239, 543)

Let's have a look at the out of bag MCC, to make sure that there is no grievous loss of performance:

julia
outofbag(ensemble) |> mcc
0.6467909354946678

Because the predictions are returned as layers, we can plot them directly:

Code for the figure
julia
f = Figure(; size = (600, 600))
ax = Axis(f[1, 1]; aspect = DataAspect(), title = "Prediction")
hm = heatmap!(ax, prd; colormap = :linear_worb_100_25_c53_n256, colorrange = (0, 1))
Colorbar(f[1, 2], hm)
ax2 = Axis(f[2, 1]; aspect = DataAspect(), title = "Uncertainty")
hm =
    heatmap!(ax2, quantize(unc); colormap = :linear_gow_60_85_c27_n256, colorrange = (0, 1))
Colorbar(f[2, 2], hm)

Partial responses and explanations ​

Like predictions, asking for a partial response or a Shapley value with a vector of layers will return the output as layers, with the value calculated for each pixel:

julia
part_v1 = partialresponse(sdm, layers, 1; threshold = false);
shap_v1 = explain(sdm, layers, 1; threshold = false, samples = 50);

We can confirm that these two approaches broadly agree about where the effect of the first variable (temperature) is leading the model to predict presences:

Code for the figure
julia
f = Figure(; size = (600, 600))
ax = Axis(f[1, 1]; aspect = DataAspect(), title = "Shapley values")
hm = heatmap!(
    ax,
    shap_v1;
    colormap = :diverging_gwv_55_95_c39_n256,
    colorrange = (-0.3, 0.3),
)
Colorbar(f[1, 2], hm)
ax2 = Axis(f[2, 1]; aspect = DataAspect(), title = "Partial response")
hm = heatmap!(ax2, part_v1; colormap = :linear_gow_65_90_c35_n256, colorrange = (0, 1))
Colorbar(f[2, 2], hm)

We can also get the Shapley values for all layers, which will return a vector of layers, one for each included variable:

julia
S = explain(sdm, layers; threshold = false, samples = 100);

We can then put this object into the mosaic function to get the index of which variable is the most important for each pixel:

Code for the figure
julia
f = Figure(; size = (600, 300))
ax = Axis(f[1, 1]; aspect = DataAspect())
heatmap!(
    ax,
    mosaic(argmax, S);
    colormap = cgrad(
        :isoluminant_cgo_80_c38_n256,
        length(variables(sdm));
        categorical = true,
    ),
)
contour!(ax, predict(sdm, layers); color = :black, linewidth = 0.5)
lines!(ax, CHE.geometry[1]; color = :black)
hidedecorations!(ax)
hidespines!(ax)
