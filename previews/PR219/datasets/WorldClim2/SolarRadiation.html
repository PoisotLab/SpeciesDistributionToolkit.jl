SolarRadiation

The SolarRadiation dataset is provided as part of the WorldClim2 provider. For more information about this dataset, please refer to: https://www.worldclim.org/data/index.html

To access this dataset:

using SpeciesDistributionToolkit
layer = SimpleSDMPredictor(RasterData(WorldClim2, SolarRadiation)) # You will probably need keyword arguments here

The remainder of this page will list the keywords you can use to retrieve specific months, layers, etc.

Layers

This dataset has no support for layers.

Resolutions

The following resolutions are accessible through the resolution keyword argument:

ResolutionKey
5m5.0
30s0.5
10m10.0
2.5m2.5

You can also list the resolutions using SimpleSDMDatasets.resolutions(RasterData{WorldClim2, SolarRadiation}).

Additional keyword arguments

This dataset has no non-standard keywords arguments.

Months

This dataset can be accessed monthly, using the month keyword argument.

You can list the available months using SimpleSDMDatasets.months(RasterData{WorldClim2, SolarRadiation}).
