HabitatHeterogeneity

The HabitatHeterogeneity dataset is provided as part of the EarthEnv provider. For more information about this dataset, please refer to: https://www.earthenv.org/texture

To access this dataset:

using SpeciesDistributionToolkit
layer = SimpleSDMPredictor(RasterData(EarthEnv, HabitatHeterogeneity)) # You will probably need keyword arguments here

The remainder of this page will list the keywords you can use to retrieve specific months, layers, etc.

The following resolutions are accessible through the resolution keyword argument:

ResolutionKey

You can also list the resolutions using SimpleSDMDatasets.resolutions(RasterData{EarthEnv, HabitatHeterogeneity}).

Additional keyword arguments

This dataset has no non-standard keywords arguments.

Months

This dataset is not indexed by months
