AverageTemperature

The AverageTemperature dataset is provided as part of the CHELSA1 provider. For more information about this dataset, please refer to: https://chelsa-climate.org/

To access this dataset:

using SpeciesDistributionToolkit
layer = SDMLayer(RasterData(CHELSA1, AverageTemperature)) # You will probably need keyword arguments here

The remainder of this page will list the keywords you can use to retrieve specific months, layers, etc.

Layers

This dataset has no support for layers.

Resolutions

This dataset is provided in a single resolution

Additional keyword arguments

This dataset has no non-standard keywords arguments.

Months

This dataset can be accessed monthly, using the month keyword argument.

You can list the available months using SimpleSDMDatasets.months(RasterData{CHELSA1, AverageTemperature}).
