MinimumTemperature

The MinimumTemperature dataset is provided as part of the CHELSA2 provider. For more information about this dataset, please refer to: https://chelsa-climate.org/

To access this dataset:

using SpeciesDistributionToolkit
layer = SimpleSDMPredictor(RasterData(CHELSA2, MinimumTemperature)) # You will probably need keyword arguments here

The remainder of this page will list the keywords you can use to retrieve specific months, layers, etc.

Layers

This dataset has no support for layers.

Resolutions

This dataset is provided in a single resolution

Additional keyword arguments

This dataset has no non-standard keywords arguments.

Months

This dataset can be accessed monthly, using the month keyword argument.

You can list the available months using SimpleSDMDatasets.months(RasterData{CHELSA2, MinimumTemperature}).

Support for future scenario SSP126

Note that the future scenarios support the same keyword arguments as the contemporary data.

Models: GFDL_ESM4, IPSL_CM6A_LR, MPI_ESM1_2_HR, MRI_ESM2_0 and UKESM1_0_LL

Timespans: Year(2011) => Year(2040), Year(2041) => Year(2070) and Year(2071) => Year(2100)
