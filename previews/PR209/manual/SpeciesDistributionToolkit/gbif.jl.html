SimpleSDMLayers.mask! — Function
mask!(l1::T1, l2::T2) where {T1 <: SimpleSDMLayer, T2 <: SimpleSDMLayer}

Changes the second layer so that the positions for which the first layer is false (for Boolean layers) or nothing are set to nothing. This is mostly useful in cases where you have a Bool layer.
