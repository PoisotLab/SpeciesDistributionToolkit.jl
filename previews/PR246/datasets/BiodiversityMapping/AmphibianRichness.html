AmphibianRichness

The AmphibianRichness dataset is provided as part of the BiodiversityMapping provider. For more information about this dataset, please refer to: https://biodiversitymapping.org/

To access this dataset:

using SpeciesDistributionToolkit
layer = SDMLayer(RasterData(BiodiversityMapping, AmphibianRichness)) # You will probably need keyword arguments here

The remainder of this page will list the keywords you can use to retrieve specific months, layers, etc.

Layers

The following layers are accessible through the layer keyword:

Layer codeDescription
GymnophionaGymnophiona
AmphibiansAmphibians
AnuraAnura
CaudataCaudata

Resolutions

This dataset is provided in a single resolution

Additional keyword arguments

This dataset has no non-standard keywords arguments.

Months

This dataset is not indexed by months
